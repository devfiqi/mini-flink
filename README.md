# mini-flink

A learning project for Apache Flink + Kafka on Kubernetes.

## What I'm Learning
- Flink sliding window aggregation
- Kafka producer/consumer groups
- Deploying Flink on Kubernetes via the Flink K8s Operator
- Kafka on K8s via Strimzi Operator

## Stack
| Layer | Tech |
|---|---|
| Stream Processing | Apache Flink 1.18 |
| Messaging | Kafka (Strimzi) |
| Orchestration | Kubernetes (minikube) |
| Job Language | Java 11 |
| Producer/Consumer | Python 3.11 |
